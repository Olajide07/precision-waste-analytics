**Precision Waste: Institutional Analytics & Forecasting**
**Project Overview:** This repository contains a full-stack data engineering and predictive analytics solution designed to modernise university waste management. By migrating legacy Excel datasets into a relational PostgreSQL environment, the project establishes a "Single Source of Truth" for environmental reporting.
The core of the project is a seasonal ARIMA (sARIMA) model that provides a 12-month predictive baseline for the 2025/26 academic session, enabling proactive budgeting and verified Scope 3 carbon emissions auditing.
**Key Features** : Automated ETL Pipeline: Cleans and imputes 8 years of fragmented institutional waste data.
**Carbon Intelligence**: Tracks a 74% reduction in carbon footprint ($12.94$ to $3.32$ $tCO_2e$) through activity-based reporting.Interactive Dashboard: Live connection to Google Looker Studio for real-time stakeholder monitoring.
**Tech StackLanguages:** Python (Pandas, Statsmodels, SQLAlchemy) **Database**: PostgreSQL. **Visualisation**: Google Looker , **StudioEnvironment**: Google Colab
**Results Summary2025/26 Baseline**: Predicted peak of ~92,000 kg/month.Accuracy: RMSE of 10,176.49 (representing high reliability for institutional planning).
**Impact**: Transitioned the university from spend-based estimates to verified activity-based carbon reporting.
